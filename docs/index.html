Overview

simple_cairo is a production-ready 2D graphics library for Eiffel that wraps the Cairo graphics library. It provides a fluent API for drawing shapes, text, gradients, and waveform visualizations with full Design by Contract support.

Part of the Simple Eiffel ecosystem.

Quick Start

Installation

  1. Install Cairo library on your system (see cairographics.org)
  2. Set the environment variable: 
    export SIMPLE_EIFFEL=/d/prod
  3. Add to your ECF file: 
    <library name="simple_cairo" location="$SIMPLE_EIFFEL/simple_cairo/simple_cairo.ecf"/>

Basic Drawing

local
    cairo: SIMPLE_CAIRO
    surface: CAIRO_SURFACE
    ctx: CAIRO_CONTEXT
do
    create cairo.make
    surface := cairo.create_surface (400, 300)
    ctx := cairo.create_context (surface)

    -- Clear to white
    ctx.set_color_rgb (1.0, 1.0, 1.0).paint.do_nothing

    -- Draw blue rounded rectangle
    ctx.set_color_hex (0x3498DB)
       .rounded_rectangle (50, 50, 300, 200, 20)
       .fill.do_nothing

    -- Draw red circle outline
    ctx.set_color_hex (0xE74C3C)
       .set_line_width (4.0)
       .stroke_circle (200, 150, 50).do_nothing

    -- Save to file
    surface.write_png ("output.png")

    ctx.destroy
    surface.destroy
end

Key Features

Fluent API

Chain drawing operations naturally with method chaining and .do_nothing termination.

Surface Types

ARGB32, RGB24, A8, A1 image formats for various use cases.

Shape Drawing

Rectangles, rounded rectangles, circles, lines, arcs, and Bezier curves.

Gradients

Linear and radial gradients with multiple color stops.

Text Rendering

Font selection, sizing, and anti-aliased text rendering.

Waveform Visualization

Audio waveform drawing from PCM samples for speech applications.

Transformations

Translate, scale, rotate operations on the drawing context.

API Reference

SIMPLE_CAIRO (Facade)

Factory class for creating surfaces, contexts, and gradients.

create_surface (width, height: INTEGER): CAIRO_SURFACE
create_context (surface: CAIRO_SURFACE): CAIRO_CONTEXT
vertical_gradient (y1, y2: REAL_64): CAIRO_GRADIENT
horizontal_gradient (x1, x2: REAL_64): CAIRO_GRADIENT
radial_gradient (cx, cy, radius: REAL_64): CAIRO_GRADIENT

CAIRO_SURFACE

Image surface wrapper with PNG export.

width: INTEGER
height: INTEGER
write_png (filename: STRING): BOOLEAN
destroy

CAIRO_CONTEXT

Drawing context with fluent API. All drawing methods return Current for chaining.

set_color_rgb (r, g, b: REAL_64): CAIRO_CONTEXT
set_color_rgba (r, g, b, a: REAL_64): CAIRO_CONTEXT
set_color_hex (color: NATURAL_32): CAIRO_CONTEXT
fill_rect (x, y, w, h: REAL_64): CAIRO_CONTEXT
stroke_rect (x, y, w, h: REAL_64): CAIRO_CONTEXT
fill_circle (cx, cy, radius: REAL_64): CAIRO_CONTEXT
stroke_circle (cx, cy, radius: REAL_64): CAIRO_CONTEXT
rounded_rectangle (x, y, w, h, radius: REAL_64): CAIRO_CONTEXT
line (x1, y1, x2, y2: REAL_64): CAIRO_CONTEXT
set_line_width (width: REAL_64): CAIRO_CONTEXT
set_gradient (grad: CAIRO_GRADIENT): CAIRO_CONTEXT
paint: CAIRO_CONTEXT
fill: CAIRO_CONTEXT
stroke: CAIRO_CONTEXT
draw_waveform_i16 (data: POINTER; count, x, y, w, h: INTEGER): CAIRO_CONTEXT

CAIRO_GRADIENT

Gradient pattern with color stops.

add_stop_rgb (offset, r, g, b: REAL_64): CAIRO_GRADIENT
add_stop_rgba (offset, r, g, b, a: REAL_64): CAIRO_GRADIENT
add_stop_hex (offset: REAL_64; color: NATURAL_32): CAIRO_GRADIENT
destroy

Gradients Example

local
    grad: CAIRO_GRADIENT
do
    -- Create vertical gradient
    grad := cairo.vertical_gradient (0, 100)
    grad.add_stop_hex (0.0, 0x3498DB)   -- Blue at top
        .add_stop_hex (1.0, 0x2ECC71)   -- Green at bottom
        .do_nothing

    -- Fill rectangle with gradient
    ctx.set_gradient (grad)
       .fill_rect (0, 0, 200, 100).do_nothing

    grad.destroy
end
